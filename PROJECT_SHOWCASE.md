# APEX: AI Fitness & Nutrition Coach - Project Showcase

## 🚀 What is APEX?
APEX is a highly personalized AI fitness coach and nutritionist web application. It is designed to track a user's full history, stats, and preferences across every session to provide deeply personalized and intelligent coaching, specifically tailored to their body, lifestyle, and goals.

## 🏆 What We Have Achieved

Through this project, we have successfully built a full-stack, AI-powered application from the ground up. Here are the core accomplishments:

1. **Intelligent AI Coach Integration (`backend/services/ai.py`)** 🧠
   - Successfully integrated the Anthropic Claude API to serve as a conversational AI coach.
   - The AI agent is context-aware: it dynamically factors in the user's recent workouts, meals, active plans, and demographic profile to give highly specific, non-generic advice.

2. **Advanced Logging & Tracking System** 🏋️‍♂️🥗
   - Developed robust forms for users to quickly and efficiently track multi-set, custom workouts.
   - Implemented detailed macro-level nutrition logging to keep track of daily dietary intake.

3. **Dynamic AI Plan Generator** 📅
   - Engineered a generative feature that allows users to express their fitness aspirations, prompting the AI to architect persistent 4-week workout routines and tailored nutrition diets.

4. **Comprehensive User Dashboard** 📊
   - Built an interactive, at-a-glance dashboard mapping tracked metrics, workout streaks, and 7-day data visualizations (using Recharts) for calories, protein, and overall activity.

5. **Modern, Scalable Full-Stack Architecture** 🛠️
   - **Frontend:** Crafted a highly responsive and fluid UI using React 18, Vite, and Tailwind CSS. Managed complex application state gracefully with Zustand, and utilized Recharts for engaging data visualization.
   - **Backend:** Designed a remarkably fast and reliable RESTful API using Python 3.11, FastAPI, and SQLAlchemy, powered by local SQLite for seamless setup.
